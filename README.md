
# KeenKeeper — Keep Your Friendships Alive

A friendship tracking web application that helps users maintain meaningful relationships by managing contacts, tracking interactions, and monitoring friendship goals.

---

## 🚀 Technologies Used

- Next.js
- React.js
- Tailwind CSS
- Recharts
- React Icons
- React Hot Toast

---

## ✨ Key Features

- 👫 View detailed friend profiles with relationship status and contact goals
- 📜 Track Call, Text, and Video interactions through a dynamic timeline system
- 📊 Visualize friendship analytics using interactive charts and filters
